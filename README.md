# 📊 Auto Financial & Product Performance Pack Generator

Automated PowerPoint generator built using Python.

## Features
- Revenue & Profit Trend
- YOY Growth Analysis
- Budget vs Actual
- Expense Breakdown
- YTD Summary
- Product Revenue Comparison
- Product Profit Comparison
- Product Margin Analysis
- Product YOY Growth %

## Tech Stack
- Python
- Pandas
- python-pptx

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Place CSV files in project folder

3. Run:
   python generate_financial_pack.py

## Output
Auto-generated PowerPoint Financial Pack
